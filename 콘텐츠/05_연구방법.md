# 05. 연구 방법 (Method)

## 연구 3단계 흐름

```
① 전환임계점 분석 → 건폐율 구간 산정        [완료]
   고정 용적률 250%하에서 건폐율 변화에 따른
   주거 성능·법적 제약 임계치 도출
        ↓
② 각 구간별 자동배치안 생성                 [진행]
   시나리오 A/B/C별로 법규·건축 요소를 반영해
   배치안 자동 생성 (Rhino/GH/Python)
        ↓
③ 평가지표로 비교·평가
   향·규제 타협에 따른 가로공간 성능 변화
   정량 분석 (지표 10개)
```

## 변인 설정

| 구분 | 항목 |
|---|---|
| **조작변인** | 건폐율 (5 / 10 / 20 / 30 / 40 / 50 %) — 독립 설정 |
| **통제변인** | 대지 면적·형상, 용적률 250%(연면적), 주동 유닛 16종, 층고 3,000mm |
| **종속변인 (1단계, 형태적)** | 건축물 층수, 동수, 인동간격 |
| **종속변인 (2단계, 성능적)** | 시각·가로공간 성능 지표 (A3, A4, A5, A6, B2, B3, C1, C2 …) |

## 건폐율 기준 (서울시 조례)
| 용도지역 | 용적률 | 건폐율 |
|---|---|---|
| 1종 일반주거 | 100~150% | 60% |
| 2종 일반주거 | 100~200% | 60% |
| 3종 일반주거 | 100~250% | 50% |

→ **용적률 250% 고정, 건폐율 상한 60%** 를 분석 범위로 설정.

건폐율 케이스 의미:
- **5%**: 49층급 초고층 특수 케이스
- **10·20·30·40%**: 변화 구간
- **50%**: 3종 일반주거 상한
- **60%**: 1·2종 상한 (일부 유닛은 도달 전 한계)

## 방법론 요약
- 동일 밀도(FAR 250%)에서 건폐율만 바꿔 배치안을 자동 생성
- 보행자 눈높이(1.6m) 관측점 기준으로 가로 성능 측정
- 시나리오 A→B→C로 향·법규 타협 정도를 단계적으로 비교
