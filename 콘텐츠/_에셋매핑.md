# 에셋 매핑 (제작용)

> 웹에 쓸 이미지/도면/영상 소스와 변환 필요 작업 정리.

## 즉시 사용 가능 (이미 PNG/JPG)
| 용도 | 파일 |
|---|---|
| Hero/매스 렌더 | `3. 매스 분석/매스 뷰/Scene 1.png`, `Scene 3.png` |
| 도시 단면 렌더 | `99. 중크/0502/Image.png`, `Image(1).png` |
| SVF/AO 분석 렌더 | `99. 중크/0502/Image_SkyMask.png`, `Image(1)_AO.png` |
| 앙각 검증 프리뷰 | `4. 배치 분석/지표별/_조망_검증preview.png` |
| 조망점 프리뷰 | `3. 매스 분석/_조망점_preview.png` |
| 문제의식 | `0. 주차별/0108/건축설계7_아파트(외부인/내부인).png`, `서론 이미지.jpg` |
| 사례 답사 | `0. 사례분석/답사 사례_embedded_files/*.png` |

## 변환 필요
| 작업 | 대상 | 방법 |
|---|---|---|
| .ai → SVG/PNG | 다이어그램류 (`중크_*.ai`, `사이트 역사.ai`, `입면 타이폴로지.ai` 등) | Illustrator 내보내기 |
| .3dm → PNG 렌더 | 건폐율 5~50% 배치안, 매스 유형화 | Rhino 렌더/캡처 |
| .pdf → 썸네일 | 주차별 발표, 참고논문 표지 | 첫 페이지 이미지화 |
| .mp4 임베드 | `건축설계7(01)분석현황.mp4` | HTML5 video |

## 폴더 구조 제안 (웹)
```
웹사이트/
├─ 콘텐츠/            (이 md들)
├─ site/             (실제 웹사이트)
│  ├─ index.html
│  ├─ results.html
│  ├─ metrics.html
│  ├─ process.html
│  ├─ references.html
│  ├─ css/style.css
│  ├─ js/main.js
│  └─ assets/
│     ├─ img/        (변환된 png/svg/jpg)
│     └─ video/
```
