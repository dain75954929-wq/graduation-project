# 06. 유형 분류 (Typology)

대상 주동·유닛·배치를 체계적으로 분류해 통제변인(16종 유닛)을 정의한다.

## 1) 매스 타입 (Mass Type)

### 일반 분류
- Plate type (판상형)
- Tower type (탑상형)
- Mixed type (혼합형)

### 기하 형태
- I자형 · L자형 · T자형 · Y자형 · X자형 · ㅁ자형

### 변형 단계
| 단계 | 명칭 | 특징·시기 | 주요 유형 |
|---|---|---|---|
| 1단계 | 원형 | 2000년 이전 단순 기하 형태 | I0, L0, T0, Y0 |
| 2단계 | 1차 변형 | 원형 유지 + 일부 단순 변형 | I1, L1, X1, ㅁ1 |
| 3단계 | 2차 변형 | 2000년대 이후 Y·ㅁ 결합 복합 | Lx2, Lㅁ2, Tㅁ2, Iㅁ2, Yㅁ2 |
| 4단계 | 3차 변형 | 탑상+판상 결합·확장·제거 고도 변형 | Lx3, Yㅁ3 |

## 2) 유닛 타입 (Unit Type)
- **개방 타입**: 1side / 2side(맞통풍 여부) / 3side
- **세대 조합**: 2세대 / 3세대 / 4세대 …
- **코어 타입**: Hall type / 계단실형(Staircase-adjacent) / 혼합형

## 3) 배치 타입 (Arrangement Type)
- **동 간 관계**: Single / Parallel / Latticed / Staggered / Mixed
- **동–가로 관계**: 수직·수평 (Type1) / 사선 (Type2)
- **동–내부공간 관계**: 중심형 (Type C) / 균등분산형 (Type U)
