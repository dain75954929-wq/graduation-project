## A. 물리적 형태

### A3 — 최고/평균 높이
건폐율 변화에 따른 주동 평균 높이. 수직 규모 정량화. (`지표_A2_높이.py`)

대지 내 1층 건축면적 합 / 대지면적. 핵심 독립변수. (`지표_A1_BCR.py`)

### A5 — 동 수

## B. 가로변 보행자 인지

### B1/B2 — 앙각 (Elevation Angle, α)
보행자 관측점에서 매스 top edge를 올려다보는 수직 각도.
```
α = atan(H / D)
H = top_z − eye_z (관측점 위로 본 매스 높이)
D = 매스 wall까지 수평거리
```
| α | 의미 |
|---|---|
| 0° | horizon |
| 45° | D = H |
| 90° | 매스 바로 위 |

**측정 방식 옵션**
| 옵션 | 정의 |
|---|---|
| A 최근접 매스 | 가장 가까운 wall의 α (검증용) |
| B FOV 내 평균 | 시야각(수평120°×수직60°) ray들의 α 평균 (현재 코드) |
| C 360° 평균 | 전방위 평균 |
| D cone 평균 | 시선 정면 cone 내 평균 |

현재 코드: **옵션 B** (`_metric_core.py`의 `compute_fov_3d_at_point`)

**판정 기준**
| α | 도시 경험 | D/H |
|---|---|---|
| ≤27° | 광장형 | ≥2 |
| 27~45° | 편안(최적) | 1~2 |
| 45~63° | enclosure 강 | 0.5~1 |
| 63~80° | 답답 | <0.5 |
| >80° | 수직 압박 | — |

> 비고: 5% 시나리오는 매스가 50층급(평균140m)이라 어느 방향이든 α가 cap(60°)에 닿아 시나리오 구분력이 약함 → 관측 기준 재정의 검토 중.

### B3 — SVF (Sky View Factor)
보행자 시점 천공 가시율 (입체각 기반, 높이·폭·거리 종합).
- 기준: 0~8% 견딜 만함, 8% 초과 심리적 압박 (Asgarzadeh 2012, 분모 2π sr)
- (`지표_B2_입체각.py`, `지표_B5_조망앙각.py`)
